# VistaVise Consulting

Production website for VistaVise Consulting.

The active product is the public Next.js consulting website in `frontend/`.
The optional Express backend in `backend/` only exposes a health check.

## Run frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend URL: `http://localhost:3000`

## Run backend

```bash
cd backend
npm install
npm run dev
```

Backend default URL: `http://localhost:4000`
